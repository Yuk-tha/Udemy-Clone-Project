# 🎓 Udemy Clone

A responsive front-end clone of the Udemy website built to replicate its core UI and layout. This project demonstrates strong fundamentals in modern web design, focusing on clean structure, responsive layouts, and user-friendly interface design.

## 🚀 Features

* Responsive navbar with search bar
* Category menu section
* Hero banner with background image
* Course cards with hover effects
* Recommended topics section
* Popular courses grid layout
* Structured footer

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* Google Fonts
* Font Awesome

## 🎯 Outcome

Gained hands-on experience in building real-world UI layouts and improving UI/UX design skills.
